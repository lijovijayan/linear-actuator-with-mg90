# Linear Actuator with MG90S Micro Servo

A compact linear actuator mechanism using a rack and pinion gear system, designed to work with the Tower Pro MG90S micro servo. Perfect for small robotics projects, automation tasks, and DIY mechanisms.

## Features

- **Compact Design**: Small footprint suitable for space-constrained projects
- **MG90S Compatible**: Designed specifically for the popular Tower Pro MG90S micro servo
- **Rack and Pinion System**: Converts rotational motion to linear motion efficiently
- **Easy Integration**: Available in STEP and F3Z formats for compatibility with most CAD software
- **Mounting Points**: Includes holes for secure mounting in your projects

## Usage

1. **Download** the CAD file (STEP for universal compatibility, F3Z for Fusion 360)
2. **Import** into your preferred CAD software (Fusion 360, SolidWorks, FreeCAD, etc.)
3. **Integrate** the design into your project
4. **3D Print** or manufacture the rack and gear components
5. **Assemble** with your MG90S servo

## Hardware Requirements

- Tower Pro MG90S Micro Servo (or compatible)
- 3D printed or machined rack and pinion components
- Mounting hardware (screws, nuts as needed for your application)
